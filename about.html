Escolhemos esse jogo pelo fato dele ser uma demonstração minimalista,
podendo servir de base para o desenvolvimento de outros jogos nesse tipo de plataforma.
Utilizamos apenas HTML, CSS e Javascript. Também usamos a framework Bootstrap.
Usamos como base o livro 'Eloquent Javascript', de Marijn Haverbeke,
para escrever o código do jogo.

A atividade foi realizada durante o Bootcamp Mastertech em Janeiro de 2018.
O Bootcamp é uma experiência de aprendizado totalmente
imersiva, que acontece durante dois meses
e em tempo integral.
